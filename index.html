MANUAL DE HTML5 Y CSS3

Empezar el html

<!-- --> - Para comentar una o mas líneas
<!DOCTYPE> - Doctype es el encargado de indicar al navegador web que el documento que va a abrir es html.
<html></html> - Dentro de esta etiqueta vamos a poner todo nuestro código de html.
  • lang="es" --> se pone para decir que el documento es en español
<head></head> - Información sobre el documento. Colección de metadatos acerca del documento.
  • <title>title --> Título de nuestro html<title>
  • <meta charset="UTF-8"> --> Indicamos la codificación
  • <meta name="author" content="[Nombre del creador]"> --> Indicamos el nombre del autor del html
  • <link rel="stylesheet" type="text/css" href="estilos.css"> --> Indicamos el donde está nuestro css
<body></body> - Contenido del documento. Solo puede haber un body en el documento

Titulos

<hn></hn>
  • h1,h2....h6
  • h1 es la letra más grande y h6 es la más pequeña
<article></article> - Composición independiente al documento
<section></section> - Determina que contenido corresponde a qué parte
<aside></aside> - Sección que se relaciona con la página pero que si la eliminamos nada pierde su sentido
<pre></pre> - Texto preformateado. La fuente no cambia. Espacios, tabuladores... Todo lo que está dentro del "pre" lo muestra como está escrito
<p></p> - Indica párrado. Distribución del texto en párrados
<nav></nav - Enlaces de navegación
<header></header> - Cabecera de la página. Aquí pondremos el título, navegación de contenidos..
<footer></footer> - Pie de página
<address></address> - Información de contacto
<li></li> - Lo usamos para poner cada elemento de las listas "ul" o "ol". li es un elemento vacío, así que no hace falta cerrarlo.
<ul></ul> - Listas desordenadas. Puede darse los siguientes valores: (Si no ponemos nada sale por defecto dependiendo del anidamiento)
  • disc
  • Square
  • circle
<ol></ol> - Listas ordenadas: (Formato 1,a,A,I,i)
  1. Números naturales
  2. Letras en minúculas
  3. Letras en MAYÚSCULAS
  4. Número romanos en MAYÚSCULAS
  5. Números romanos en minúculas
- Usamos reversed en la etiqueta ol.
  1. Para mostrar las listar en orden invertido
  2. Otra línea para la muestra
- Usamos start para empezar en un número o letra que queramos
  1. Para empezar la lista en el número indicado
  2. Otra línea para la muestra
